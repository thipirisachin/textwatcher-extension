Extension setup (one-time)

  1. 1 Open chrome://extensions → enable Developer mode → click Load unpacked → select C:\Projects\textwatcher-extension
  2. 2 Windows Settings → System → Notifications → make sure Google Chrome notifications are On
  3. 3 Click the TextWatcher toolbar icon → add URL rule http://localhost:8080/* (type: Wildcard)
  4. 4 Add keyword TEXTWATCHER_SIGNAL (type: Contains, both alert toggles ON)
  5. 5 Reload this tab (Ctrl+R) so the content script gets injected, then click Show Signal below → you should get a desktop notification within 1–2 seconds
